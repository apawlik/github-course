Workshop information

This hands-on workshop will provide introduction on using GitHub for collborative work. Most of the material is based on the Software Carpentry Git lesson. This workshop is intended for anyone who wants to learn how to use GitHub - you don't need to know anything about version control before coming to this session. We will only learn how to use GitHub, please note that we will not cover using Git on your laptop.

Requirements: Participants must bring a laptop with a Mac, Linux, or Windows operating system (not a tablet, Chromebook, etc.). They are also required to abide by Software Carpentry's Code of Conduct (which we refer back to as a well working model).

Setup

Git

Git is a version control system that lets you track who made changes to what when and has options for easily updating a shared or public version of your code on github.com. You will need a supported web browser (current versions of Chrome, Firefox or Safari, or Internet Explorer version 9 or above).

GitHub

At this workshop we will only work with GitHub (not with Git on your machines). You will need an account at github.com. Basic GitHub accounts are free. Please create a GitHub account if you don't have one already. Please consider what personal information you'd like to reveal. For example, you may want to review these instructions for keeping your email address private provided at GitHub.
